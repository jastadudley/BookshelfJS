# The Shelf - Book Recommendation Engine

## Overview

The Shelf is a browser-baised book recommendation interface built with vanilla JavaScript, HTML, and CSS. The application 
allows users to search for books by genre and returns matching results displayed as styled cards. It uses ES6 array 
methods to filter a curated book dataset, a recursiv function to render star ratings, and the Lodash library to debounce 
search input. The interface is styled with a dark, library-inspired easthetic using custom CSS and background imagery.
