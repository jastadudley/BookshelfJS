# The Shelf - Book Recommendation Engine

## Overview

The Shelf is a browser-based book recommendation interface built with vanilla JavaScript, HTML, and CSS. The application 
allows users to search for books by genre and returns matching results displayed as styled cards. It uses ES6 array 
methods to filter a curated book dataset, a recursive function to render star ratings, and the Lodash library to debounce 
search input. The interface is styled with a dark, library-inspired aesthetic using custom CSS and background imagery.
